Работа с программой

Для выбора файла с входными данными: Options -> Open, либо Ctrl+O. Файлы с данными - data1.txt и data2.txt

Для распечатки списка инцидентности в текстовый файл: Options -> Print GraphList, либо Ctrl+P

Вызов справки Help - F1

Вызов справки About - F2

Для обхода графа - кнопка BFS

Для возвращения графа в исходное состояние - кнопка Refresh
