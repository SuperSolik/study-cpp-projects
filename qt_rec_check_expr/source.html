Задание

Построить синтаксический анализатор для понятия простое выражение.

простое_выражение::=простой_идентификатор | (простое_выражение знак_операции простое_выражение)

простой_идентификатор::= буква

знак_операции:: = – | + | *

Работа с приложением

Для правильной работы приложения выражения в файле входных данных должны располагаться построчно

Выбор файла с данными для ввода - кнопка "Input file"

Выбор файла для вывода - кнопка "Output file"

По умолчанию файл для вывода - "output.txt" в директории сборки

Запуск анализа - кнопка "Analize"
