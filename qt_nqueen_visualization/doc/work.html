Работа с программой

Для задания размера доски - кнопка setBoardSize

Для установки начального ферзя - кнопка placeQueen

Для нахождения решения при текущем положении начального ферзя - кнопка SolveTask

Для нахождения всех решений - кнопка SolveAll, решения выведутся в текстовый файл

Вызов справки Help - F1, либо Options->Help

Вызов справки About - F2, либо Options->Help

Выбор текстового файла для вывода решений - Options->Choose out file
